# 🧬 DNA Sequence Analyzer

## Overview
This project is a Python-based DNA sequence analysis tool designed to perform basic computational analysis on nucleotide sequences. It focuses on extracting meaningful information from a linear biological sequence using fundamental programming constructs.
The program processes an input DNA sequence by iterating through each nucleotide and performing base composition analysis, positional mapping, and percentage calculation. It models the sequence as a string data structure and applies indexing techniques to track the exact location of each nucleotide (A, T, G, C).As identitificantion and positioning of DNA bases is important in DNA structural analysis.
Additionally, the program incorporates input validation to identify non-standard characters and ensures consistent processing by normalizing the sequence format.

## Features
* Accepts a DNA sequence as input
* Converts input to uppercase for consistency
* Calculates total sequence length
* Counts occurrences of each nucleotide (A, T, G, C)
* Tracks positions of each base in the sequence
* Identifies invalid characters
* Computes percentage composition of each base

## Process
The DNA sequence is treated as a string and analyzed using:

* Iteration through the sequence using loops
* Conditional checks to identify each base
* Lists to store positions of nucleotides
* Basic mathematical operations for percentage calculation

## Concepts Used
* Loops (`for`)
* Conditional statements (`if`, `elif`)
* Lists
* String handling
* Built-in functions (`len`, `input`)
